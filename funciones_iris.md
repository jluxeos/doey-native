# IRIS — Funciones Disponibles

## Dispositivo
| Comando ejemplo | Acción |
|---|---|
| `enciende/apaga la linterna` | Linterna ON/OFF |
| `sube/baja el volumen`, `volumen 70` | Volumen media (0–100) |
| `sube/baja el timbre` | Volumen timbre |
| `sube/baja el volumen de alarma` | Volumen alarmas |
| `silencia`, `modo vibración`, `sonido normal` | Modo silencio |
| `sube/baja el brillo`, `brillo 50`, `brillo automático` | Brillo pantalla |
| `activa/apaga wifi` | WiFi |
| `activa/apaga bluetooth` | Bluetooth |
| `activa/apaga modo avión` | Modo avión |
| `activa/apaga no molestar` | No molestar |
| `activa/apaga NFC` | NFC |
| `activa/apaga modo oscuro` | Dark mode |
| `activa/apaga hotspot` | Punto de acceso |
| `ahorro de energía` | Power save |
| `pantalla se apague en 30 segundos` | Timeout pantalla |
| `toma captura` | Screenshot |
| `bloquea el teléfono` | Bloquear pantalla |

## Notas del sistema
- Comandos con `oye`, `hey`, `iris`, `doey` al inicio se ignoran automáticamente
- `por favor` al final se ignora
- Tolerancia a variantes de voz-a-texto (guasap, flas, k, q, etc.)
- Orden flexible: `wifi activa` = `activa wifi`
- Frases con pregunta, contexto externo (carro, TV, casa) o condicionales → IA
